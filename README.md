# Portfólio Git — CIN0136

Repositório individual de evidências de aprendizado em Git, desenvolvido para
a disciplina **Desenvolvimento de Software (CIN0136)** do CIn-UFPE.

## Estrutura

```
seu-portfolio-git/
├── README.md              ← este arquivo
├── REFLEXAO.md            ← reflexão pessoal sobre o processo
├── exercicios/
│   ├── ex01-commits/      ← exemplos e guia de commits semânticos
│   ├── ex02-branches/     ← guia e exemplos de uso de branches
│   └── ex03-conflito/     ← registro da resolução de merge conflict
└── projeto/
    └── hello-world/       ← mini-projeto usado nos exercícios
```

## Sobre o repositório

Este portfólio documenta o processo de aprendizagem de Git ao longo das
Semanas 2 e 3 da disciplina. O foco está no processo — histórico de commits,
uso de branches, Pull Requests e resolução de conflitos — e não no produto final.

## Exercícios

| Exercício | Descrição |
|-----------|-----------|
| [ex01-commits](exercicios/ex01-commits/) | Prática de commits semânticos com Conventional Commits |
| [ex02-branches](exercicios/ex02-branches/) | Criação e merge de branches com propósitos distintos |
| [ex03-conflito](exercicios/ex03-conflito/) | Demonstração e resolução de merge conflict |

## Como navegar

- Comece pelo [ex01-commits](exercicios/ex01-commits/exemplos.md) para entender o padrão de commits usado
